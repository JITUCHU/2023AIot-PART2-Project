홈 화면 예시

이 프로젝트는 조도센서값과 초음파 센서의 값을 받아와서,
홈 화면을 통해 출력이 한눈에 볼수 있게끔 만드는 작업을 진행하였습니다.
조도센서 값과 초음파센서 값 이 특정 값이 되면 각 요소에 설정 된 객체가 반응하게 됩니다.
이로써 이 프로젝트를 통해 간단한 자동차 오토라이트 및 거리 감지 시스템을 구현하였고
이러한 시스템은 운전중에 안전을 강화하고 어두운 환경에서 더 나은 가시성을 제공합니다.
또한 장애물 간의 거리 감지를 통해 운전자에게 경고를 제공하여 사고를 예방할수 있습니다.

각 센서의 값 기준

조도센서 : 0~ 100 % 중 50% 이하가 되면 오토라이트 작동
초음파 센서: 앞, 뒤 두개 부착하여 앞 뒤 각각 경고등 점등
10cm 이하 이면 노란색 경고등 점등, 5cm 이하이면 주황색 경고등 점등, 3cm 이하이면 빨간색 경고등 점등.
